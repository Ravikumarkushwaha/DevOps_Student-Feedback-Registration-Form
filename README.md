📘 Student Feedback Registration Form

🎯 Task Overview
This project implements a Student Feedback Registration Form using HTML, CSS, JavaScript, Selenium, and Jenkins. It covers UI design, input validation, and automated testing.

🎯 Objective
To build a user-friendly web form, validate inputs on the client side, automate test cases using Selenium, and execute them through Jenkins.

🧩 Sub Tasks

1️⃣ HTML Form

Student Name
Email ID
Mobile Number
Department
Gender
Feedback Comments
Submit & Reset buttons
2️⃣ CSS Styling

Internal and external CSS used
Clean and responsive layout
Styled inputs and buttons
3️⃣ JavaScript Validation

Name cannot be empty
Email must be valid
Mobile number must be 10 digits
Gender selection required
Department must be selected
Feedback must contain minimum 10 words
4️⃣ Selenium Testing
Test cases implemented:

Page load verification
Valid form submission
Empty field validation
Invalid email validation
Invalid mobile number validation
Dropdown selection check
Submit and Reset button functionality
5️⃣ Jenkins Automation

Jenkins installed and configured
Freestyle job created
Project connected via local workspace
Selenium tests executed using build steps
Build status verified successfully

Form validation working correctly
Selenium test cases executed successfully

The project successfully demonstrates end-to-end development, testing, and automation of a web-based form using modern tools and practices.
